AI_OS

Artificial Intelligence Operating System

About AI_OS

AI_OS — экспериментальная операционная система нового поколения с встроенным искусственным интеллектом. Система объединяет microkernel на C++, AI‑модули, современный GPU‑интерфейс и поддержку x86_64 и ARM64.

Features

AI Assistant

AI помогает управлять системой, запускать программы и диагностировать ошибки.

Modern GUI

GPU‑ускоренный интерфейс с поддержкой тем и анимаций.

Package Manager

aipkg — система пакетов для установки приложений.

Secure System

Sandbox, Secure Boot и изоляция процессов.

System Architecture

Kernel

C++ microkernel управляет памятью и процессами.

Runtime

.NET NativeAOT, Python runtime и JavaScript engine.

GUI Layer

Desktop shell и оконный менеджер.

AI Layer

LLM‑модуль для интеллектуальных функций.
