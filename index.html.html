04 Модули системы
💾
Bootloader
ASM
UEFI-загрузчик на x86-64 ASM. Инициализация CPU, GDT, переход в 64-bit режим, загрузка ядра.
🔧
Micro-Kernel
C++
Минималистичное ядро: управление памятью (VMM/PMM), планировщик, системные вызовы.
🖥️
GPU / Display
C++
Драйвер GPU с поддержкой Vulkan. Framebuffer, KMS/DRM-аналог для AI_OS.
📁
Файловая система
C++
Собственная ФС AIFS (AI File System) с журналированием. Совместимость с ext4, FAT32, NTFS (чтение).
🌐
Network Stack
C++
TCP/IP v4/v6, WiFi, Ethernet. HTTP/HTTPS клиент встроен в ядро системы.
🎨
Window Manager
C#
.NET NativeAOT оконный менеджер с GPU-compositing. Темы, анимации, мультимонитор.
🖱️
Desktop Shell
C#
Рабочий стол, панель задач, трей, уведомления, меню приложений.
⚙️
System Settings
C#
Панель управления: сеть, дисплей, пользователи, безопасность, обновления.
🤖
AI Core Engine
Python
Встроенная LLM-модель (локальная). AI-ассистент, умный поиск, автодополнение команд, голосовой ввод.
📦
Package Manager
Python
aipkg — менеджер пакетов AI_OS. Репозиторий, зависимости, обновления, sandboxing.
🔒
Security Module
C++ + ASM
ASLR, Secure Boot, sandbox-изоляция, шифрование диска, управление правами.
🌍
Browser Engine
JS
Встроенный браузер на базе Chromium Embedded Framework + V8. Полная поддержка HTML5/CSS3/JS.
💻
Terminal + Shell
C++ + Python
AI-powered терминал. Автодополнение через LLM, объяснение команд, история с поиском.
🔄
Update System
Python
Дельта-обновления ОС. A/B-разделы для безотказного обновления. Откат при ошибке.
